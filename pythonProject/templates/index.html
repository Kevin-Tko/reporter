See below instructions on how to use this WebApp

  1. Download files from Kprinter Web
  2. Rename the downloaded Incomings xls files to mt102,mt103,mt202,mt910
  3. Rename the downloaded Outgoings xls files to mt103out,mt202out,mt900out:
  4. Choose each file and Upload You cannot upload multiple
  5. Click Generate Report
  6. Click Clear to clear existing files
